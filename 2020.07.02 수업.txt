numpy 개요

수치해석용 파이선 패키지로 numerical python의 줄임말
다차원의 배열 자료구조 클래스인 ndarray(n차원의 array을 줄임말) 클래스를 지원
백터와 행렬을 사용하는 선형대수 계산 사용.(1차원~n차원을 활용하기 위해)

numpy 특징

배열 연산은 c로 구현된 내부 반복문을 사용
파이썬 반복문에 비해 빠른 속도 (for문을 줄이기 위해 활용.)
백터화 연산을 이용(백터 차원의 연산을 편하게 배열 처럼 만들기 위해 )
간단한 코드로도 복잡한 선형 대수 연산을 수행
배열 인덱싱(array indexing)을 사용한 질의(query) 가능.
 개요

배열에서 데이터 변경,정제,부분집합,필터링의 빠른수행
정렬 유일 원소 찾기 집합 연산
토예 표현과 데이터의 수집/요약
여러 데이터의 병합, 데이터 정렬과 데이터 조작
